return

return passes the control from the current function to the calling function. If the protocol is established to return a value from the called function, it can return a value to the calling function or method. This is discussed in more detail in a future chapter.

labeled statement

The labeled statement addresses the issues with using control flow statements when multiple loop structures are embedded within each other. Like if a switch is embedded in a loop, if you wanted to break the flow using 'break' statement, depends on where the line is it may skip the current switch or for loop.
